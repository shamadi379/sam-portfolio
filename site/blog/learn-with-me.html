Microsoft 365 — an introduction

Published: Oct 2025

Microsoft 365 (M365) is the cloud productivity platform that combines Office apps, identity, device management and collaboration services into a single operational model. In this Learn with Me series I’ll focus on practical admin tasks, automation, and security patterns that help keep M365 services reliable and manageable.

What I’ll cover

  • Identity & access — Entra ID (Azure AD) fundamentals, conditional access, and identity governance.
  • Collaboration — Exchange Online, Teams, SharePoint and how to manage mailbox and calendar permissions safely.
  • Device & app management — Intune basics, application deployment, and endpoint compliance.
  • Automation — PowerShell and Graph automation for repetitive tasks, bulk updates, and runbooks.
  • Backup & continuity — Practical approaches for protecting Exchange and SharePoint content; recovery drills.
  • Security & monitoring — Defender for Office 365, auditing, alerting and sensible monitoring using Sentinel/Log Analytics.

How these posts will help you

Each post will include clear commands, example scripts, and short checklists so you can apply the steps in a lab or production environment safely. Expect walkthroughs for common administrative tasks (for example: granting calendar permissions, bulk updates, or automating license assignments).

Where to start

If you’re interested in Exchange topics, click the Exchange category. For identity and tenant-wide items, check the M365 category. For infrastructure automation using ARM/CLI, visit Azure.

I’ll keep content short and practical — if there’s a topic you’d like covered, tell me and I’ll add it to the queue.
